# GameBoy

## Games Included

### Snake
- Dynamic grid-based movement
- Collision detection (walls + self)
- Real-time score tracking

### Wordle
- Word-guessing game with color-coded letter feedback
- Randomized word selection
- Input validation

### Hangman
- Classic word-reveal gameplay
- Classic word-reveal gameplay
- Graphical visuals rendered via SFML

## Tech Stack
- Language: C++
- Graphics Library: SFML 
- Paradigm: Object-Oriented Programming (OOP)
- Interface: Graphical (SFML window)
